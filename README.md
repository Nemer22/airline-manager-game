# 🛫 Airline Manager - إدارة الخطوط الجوية المتقدمة

لعبة محاكاة إدارة خطوط جوية متقدمة باللغة العربية مع نظام ألعاب عميق وشامل.

## ✨ المميزات الرئيسية

### 1️⃣ نظام الوقود والصيانة المتقدم
- حساب استهلاك وقود واقعي لكل طائرة
- نظام تعبئة الوقود مع تكاليف متغيرة
- جدول صيانة دوري للطائرات
- تأثير الصيانة على أداء الطائرة

### 2️⃣ نظام السمعة والتصنيف
- تقييم الشركة من 1-5 نجوم
- تأثير السمعة على الأرباح والعملاء
- رحلات موفقة = زيادة السمعة
- تأخيرات وحوادث = انخفاض السمعة

### 3️⃣ نظام الأرباح الديناميكي
- أسعار رحلات تتغير حسب الطلب
- موسمية (أوقات ذروة وهادئة)
- تأثر الأسعار بسمعة الشركة
- عروض خاصة موسمية

### 4️⃣ نظام المنافسة
- شركات منافسة تقلل حصتك السوقية
- حروب أسعار ديناميكية
- استراتيجيات السوق

### 5️⃣ نظام الأبحاث والتطوير
- تحسين كفاءة الوقود (قطع غيار)
- تطوير طائرات جديدة
- تقليل وقت الصيانة
- زيادة سعة الوقود

### 6️⃣ نظام الفروع والمحاور
- فتح مقرات في مدن جديدة
- تحسين البنية التحتية
- رحلات متصلة بين الفروع

### 7️⃣ نظام الأحداث والمهام
- مهام خاصة برواتب عالية
- عقود طويلة الأجل
- أحداث عشوائية يومية
- حدث قطاع خاص لشخصيات مهمة

### 8️⃣ نظام الشراكات والتحالفات
- التحالف مع شركات أخرى
- تقاسم الرحلات (Codeshare)
- الاستفادة من الشبكات العالمية

### 9️⃣ نظام الحقبات الزمنية
- تقسيم اللعبة إلى أشهر/سنوات
- تقارير دورية عن الأداء
- أرباح نهاية الفترة

### 🔟 نظام القروض والاستثمار
- أخذ قروض بفائدة محسوبة
- الاستثمار في البنية التحتية
- شراء أسهم من الشركات

### 1️⃣1️⃣ نظام التدريب والتطور
- تدريب الطاقم يحسن الأداء
- زيادة مهارات الطيارين
- تحسين كفاءة مضيفي الكابين

### 1️⃣2️⃣ نظام الإحصائيات والرسوم البيانية
- رسم بياني لتطور الأموال عبر الزمن
- خريطة حرارية للرحلات الشهيرة
- إحصائيات مفصلة للأداء
- نظام الإنجازات

## 🎮 كيفية اللعب

1. ابدأ باللعبة برصيد أولي
2. اشترِ طائرات وطاقم
3. قم برحلات لكسب أموال
4. طور شركتك بالترقيات والبحث
5. توسع إلى مدن جديدة
6. تنافس مع الشركات الأخرى
7. اصل إلى مستويات عليا وكن الأفضل!

## 🚀 كيفية التشغيل

1. استنسخ المستودع:
```bash
git clone https://github.com/Nemer22/airline-manager-game.git
cd airline-manager-game
```

2. افتح `index.html` في المتصفح أو استخدم خادم محلي:
```bash
python -m http.server 8000
# ثم افتح http://localhost:8000
```
